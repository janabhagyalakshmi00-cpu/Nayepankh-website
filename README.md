# NayePankh Foundation Website

## Project Overview
A responsive NGO website developed for the NayePankh Foundation Internship Selection Task.

## Features
- Responsive Design
- Modern User Interface
- About Section
- Programs Section
- Volunteer Registration Form
- Contact Section
- Dark Mode Toggle
- Smooth Navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project aims to provide an engaging and informative online presence for NayePankh Foundation while encouraging volunteer participation and community engagement.
